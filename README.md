# python-job-letter

This Python program generates a job appointment letter based on user input.
It collects personal and job-related details from the user and prints a formatted employment letter.

The script demonstrates:
- User input handling
- Type conversion
- String formatting using f-strings
- Multi-line strings in Python

## Features
- Prompts the user for:
    - Name
    - Job role
    - Age
    - Salary
- Automatically formats a professional job appointment letter
- Displays the letter in the console
